CIS 127: HTML5 and CSS3

Welcome to my CIS 127 home page. This page will be the entry point for all the pages we create during the semester.

Our pages will be static HTML pages, but they could just as easily be delivered by a web framework such as Rails, Django, or PHP.

Assets

Assets include things like CSS files, images, Javascript, multimedia files. Some modern web frameworks (I'm speaking of Rails) create an asset pipeline that concatenates multiple files (CSS and Javascript) into a single minified file. The idea is to reduced the number of requests made on the server.

I suggest that you create an assets sub-directory inside of your cis127 directory. Use the assets directory to organze your site's future assets.

Validation

By the way, your page should validate if possible. Your HTML5 code may work, even be technically correct, yet show errors. Opera can validate your HTML. Right click the page and click validate.
